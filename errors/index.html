Errors

Overview

While using the sdk, you may run into an error. Usually this is caused by an invalid parameter in your implementation.


Here is an example of one error you may run into:

    { 
    Error: Invalid destination provided at ...,
    httpStatusCode: 401,
    message: 'Invalid destination provided',
    info: { unknowns: [ 'moffdd' ] } 
}

    To learn more about these and how to solve them, review the error messages below:


    Payment Errors:


    409


    Error Message:

    Request exceeds users global connect spend limit

    What happened?

    The user has reached their Connect limit.


    What can I do?



    409


    Error Message:

    Invalid destination provided

    What happened?

    The handle, paymail or address you provided is invalid


    What can I do?

    Ensure the destination you're providing is a valid one.
