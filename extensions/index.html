Extensions


What are HandCash extensions?

A HandCash extension is a developer tool used for cross-app authentication. These can be used to leverage application layers across the metaverse.

Learn how to make your own HandCash extension here.


How to use Extensions

The below example illustrates how to use HandCash extensions.

    const { HandCashConnect } = require('@handcash/handcash-connect');

const appId = '<id>'
const appSecret = '<secret>'
const authToken = '<token>'
const hasteConfig = {appId: '<id>', appSecret: '<secret>', gameId: '<id>'}

const handCashConnect = new HandCashConnect({ 
   appId: appId, 
   appSecret: appSecret,
}); 

const account = handCashConnect.getAccountFromAuthToken(token);

await account.extensions.haste.init(hasteConfig)
await account.extensions.haste.submitPlay({score: 30})
