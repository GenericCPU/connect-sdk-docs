RUN


What is RUN?

RUN is a scripting and token protocol commonly used for managing digital assets or smart contracts.


To use RUN, you must be install the RUN module

Initialize the SDK

Initialize the HandCashPurse, HandCashOwner, and Run.

    const Run = require('run-sdk');
const { HandCashPurse, HandCashOwner, Environments, } = require('@handcash/handcash-connect');

const authToken = '<authToken>';
const appSecret = '<appSecret>';


const handcashPurse = HandCashPurse.fromAuthToken(authToken, Environments.prod, appSecret );
const handcashOwner = HandCashOwner.fromAuthToken(authToken, Environments.prod, appSecret);

const run = new Run({
  purse: handcashPurse,
  owner: handcashOwner,
});

    Create an Item

    Define and deploy an Item.

      const Run = require('run-sdk');
const { HandCashPurse, HandCashOwner, Environments, } = require('@handcash/handcash-connect');

const appSecret = '<app-secret>'
const authToken = '<auth-token>';

class Weapon extends Run.Jig {
  upgrade() { this.upgrades = (this.upgrades || 0) + 1; }
  send(address) { this.owner = address; }
}

const Weapon = run.deploy(Weapon)
const sword = new Weapon();
await run.sync();

console.log(sword);
